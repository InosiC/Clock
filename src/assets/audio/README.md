# Hourly chime audio assets

Placeholder audio cues played by `useHourlyChime` at the top of every hour.

| File              | Purpose                                          |
| ----------------- | ------------------------------------------------ |
| `mickey-hour.wav` | Mickey Mouse voice clip ("Oh boy, it's o'clock!")|
| `lotr-chime.wav`  | Lord of the Rings themed musical chime           |

The committed `.wav` files are short, silent placeholders so the build and
imports resolve. Replace them with the real (licensed) audio when available,
keeping the same filenames.
